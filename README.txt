To run this project, please follow these instructions

1. Clone project
2. Create a virtual host, set up a virtual domain (example mbticheck.com) pointing to the downloaded folder.
3. Open terminal in project, cd webService and run "node index.js" to start web service.
4. Access virtual domain which is created in step 2 and add /index.html to access the website (example mbticheck.com/index.html).

It finished. Thank for reading.
